# Want Beer

Application for products delivery

## Prerequisites

- react-native
- jdk 8
- npm
- node
- yarn

## Get started

### Production release
- yarn build

### Debug release
- react-native start --reset-cache
- react-native run-android

## Development

- React-native
- Redux
- Redux-thunk
- Redux-persist
- React-navigation
- Axios
- react-native-device-info
- react-native-picker-select
